## VimChat

#### normal mode
`i` - set message mode
`:` - set command mode
`h` `j` `k` `l` - move cursor (left down up right)

#### message mode
`esc` - set normal mode
`enter` - send message

#### command mode
`esc` - set normal mode

#### multiline message mode
`esc` - set normal mode

#### commands
`:connect <url>` - connect to server
`:connect ws://localhost:8181`

`:name <name>` - set name
`:name Tester`

`:multiline` - set multiline message mode

`:send` - send message

`:exit` - exit
